## My Personal Portfolio
This is my portfolio made with React JS to show a little about me :)


## Technology 

Here are the technologies used in this project.

* React
* React Router Dom
* React-icons/AI 
* React-Social-Icons

## Services Used

* Github
* Netlify


## Getting started

* Dependency
  - Git
  - Npm

* Clone the project:
>    $ git clone https://github.com/Lucasdfg07/Reactjs-portfolio

* Run the project:
>    $ npm start
